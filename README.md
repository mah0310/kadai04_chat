# ①課題名
本の学びをあなたに送るメモアプリ

## ②課題内容（どんな作品か）
- 読んだ本をストックします。
- ボタンを押すと、過去の内容のストックからランダムで選択しあなたに送ります。

## ③アプリのデプロイURL
デプロイしている場合はURLを記入（任意）

## ④アプリのログイン用IDまたはPassword（ある場合）
なし

## ⑤工夫した点・こだわった点
- 適切に表示されるように、ストック数が足りないとアラートが出たり、タイトルだけの入力だとアラートが出るようにした。
- アイデアというのは結合によって起こるとされているので、2つ表示枠を用意した
- その2つに同じものが表示されないよう設定した

## ⑥難しかった点・次回トライしたいこと（又は機能）
- 同じものを表示させないようにする際の設定が難しかった、forでなんとかしようとしてしまい大変だった。
- 生成AIのAPIなどを用いて、出力された２つの内容から、新しいアイデアを自動で生成してくれるようにしてみたい。
